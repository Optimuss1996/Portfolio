## Tech Stack & Setup

### Technologies

- React.js
- HTML5/CSS3
- JavaScript (ES6+)
- Yarn

### Quick Start

```bash
# Install dependencies
yarn install

# Start development
yarn start

# Build for production
yarn build
```
